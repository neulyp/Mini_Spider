百度大事记
2017

  • 12月

    百度与河北省政府、雄安新区管委会签署战略合作协议，共同将雄安打造为AI-City智能城市新标杆。

  • 12月

    百度与华为、小米分别达成战略合作。在“软硬结合”的AI思维下，百度将持续发展更多合作伙伴。

  • 11月

    首批国家新一代人工智能开放创新平台名单公布，百度主导建设自动驾驶国家人工智能开放创新平台。

  • 11月

    2017百度世界大会召开，百度宣布2018年将量产无人车，同时百度首款标杆性人工智能硬件产品raven H面世。

  • 11月

    百度发布了集搜索和信息流双引擎于一体的手机百度10.0，引领新移动时代的新交互体验。

  • 10月

    百度硅谷第二个研发中心正式运营，百度西雅图研发中心也同时成立，这标志着百度在北美核心技术圈的布局初步完成。

  • 7月

    百度举办全球首个AI开发者大会，首次发布人工智能开放平台的整体战略、技术和解决方案。会上陆奇表示，百度将“All in AI”。

  • 7月

    李彦宏乘坐无人车辆开上五环，收到无人车历史上第一张罚单。四个月后，北京自动驾驶新规出台。

  • 6月

    成立不足9个月的百度信息流业务日活跃用户破亿。在搜索+信息流”双引擎强力推动下，信息流迅速成为国内又一个超级应用。

  • 6月

    百度与中国农业银行达成战略合作，并成立金融科技联合实验室。通过AI Fintech联合创新，双方将共推银行业进入智能金融新时代。

  • 5月

    百度发布新使命：用科技让复杂的世界更简单。

  • 5月

    在国务院总理李克强与德国总理默克尔见证下，百度与德国博世集团签署战略合作协议，双方将在自动驾驶、智能交通、智能车联网领域展开深入合作。

  • 4月

    百度推出Apollo（阿波罗）自动驾驶平台，向汽车行业及合作伙伴提供了一个开放、完整、安全的软件平台。这是全球范围内自动驾驶技术的第一次系统级开放。

  • 4月

    百度在人工智能领域公开的中国专利申请超过2000项，国外专利申请数百项。专利布局帮助百度抢占技术制高点。

  • 3月

    由国家发改委批复、百度牵头筹建的中国深度学习技术及应用国家工程实验室正式揭牌。

  • 2月

    《麻省理工科技评论》公布2017年十大突破技术，百度凭借领先的人脸识别技术登上榜单，包揽人工智能所有奖项。

  • 1月

    前微软全球执行副总裁陆奇加盟百度，出任百度集团总裁兼首席运营官。

  • 1月

    百度人工智能机器人小度参加《最强大脑》，对战三位人类选手二胜一平未尝败绩。
