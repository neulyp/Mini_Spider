移动获发FDD牌照，电信联通还有活路么？

丁傲西

百家号04-0320:47

文/丁傲西（本文为百家号独家）

中国移动近日开通了全球首个5G系统站点，并打通国内首个5G电话，这意味着一个万亿级产业逐步拉开序幕。此外，工信部4月3日向中国移动颁发了LTE/第四代数字蜂窝移动通信业务（LTE FDD）经营许可，即通俗所称的“4G LTE FDD牌照”。

随着“提速降费”进程的不断推进，4G网络已经成为了人们的主要选择。我们知道，中国三大运营商使用的4G网络制式并不相同。中国移动使用的是“中国自主知识产权”的TD-LTE网络，而电信和联通主要使用的是国家主流的FDD-LTE标准，TD-LTE只是作为网络的补充。而为了扶持“自主知识产权”的TD-LTE 4G标准，国家推迟了将近15个月才向电信和联通两家运营商发放了FDD-LTE牌照。而FDD牌照的姗姗来迟，也让中国移动在相当长的一段时间内完全垄断了4G市场。时至今日，电信联通两家运营商虽然奋起反击，推出了各种类型的优惠套餐，但是和移动的差距却是越来越大。中国的通信市场不仅没有出现势均力敌的竞争态势，反而呈现出严重失衡的局面。

虽然移动背负了扶持“自主知识产权”4G标准的重任，但是移动并非对FDD-LTE没有想法。一直以来，中国移动就在多个场合表达出申请FDD-LTE牌照的强烈意愿。虽然电信联通的员工们表示了强烈的反对，但是在2018年4月3日，工信部还是正式向中国移动发放了基于FDD-LTE标准的4G牌照。我们不禁要问了，移动获发FDD-LTE牌照以后，电信和联通两家运营商还有活路么？

为什么移动需要FDD-LTE？

一直以来，财大气粗的中国移动都是TD-LTE的坚定拥护者。早在TD-LTE牌照发放前，移动就已经参与了TD-LTE标准的制定，并在数十个城市部署了试验网。不仅如此，中国移动的高管还在多个场合呼吁国家同时给电信和联通发放TD-LTE牌照。虽然电信和联通两家运营商的高管也在多个场合表达了希望混合组网或者租用移动TD-LTE网络的想法，但是最终国家还是以扶持自主知识产权的理由只发放了TD-LTE牌照。由于FDD-LTE牌照的缺失，在相当长的一段时间内，财大气粗的中国移动完全垄断了4G市场，并一度占据了95%以上的4G份额。换句话说，中国移动凭借国家对TD-LTE的政策扶持，占据了4G市场的绝对优势。

然而随着移动4G用户的不断增长，同时维护2G、3G、4G三张网络需要耗费大量的人力物力。即便是财大气粗的中国移动，在巨大的网络维护成本面前也显得有限力不从心。为了降低网络维护的成本，移动一方面在全国范围内关闭了大量网络负荷较低的TD-SCDMA基站，将腾出来的频率用于TD-LTE网络的覆盖。另一方面，移动也在全国范围内开通了VoLTE技术，利用4G网络承载原本由2G、3G提供的语音业务。而VoLTE的普及，也让移动的GSM网络负荷大幅下降，移动迎来了千载难逢的2G退网时机。

然而移动的GSM网络属于FDD制式（上下行使用不同的频段），如果要在GSM频段上部署4G网络，同为FDD制式的FDD-LTE成立唯一的选择（TD-LTE上下行使用相同的频段）。而GSM网络负荷的降低，也让移动有了清频退网的可能。将清理出来的GSM频谱用于FDD-LTE网络的覆盖，不仅有助于降低网络维护成本，同时还有助于“提速降费”的进一步推动，可以说是一举两得。

电信联通会坐以待毙么？

在凭借TD-LTE成功占据了4G市场的绝对优势后，移动又把关注的重点放在了FDD-LTE上。虽然FDD-LTE对于移动有着诸多的好处，但是对于竞争对手来说，情况就并没有这么乐观了。早在几年前，正是因为只发TD-LTE牌照，电信和联通两家运营商才会迟迟无法参与4G的竞争，造成移动在4G领域的绝对垄断。一旦移动获得FDD-LTE牌照，不仅会对电信和联通的生存产生冲击，同时也会在一定程度上边缘化TD-LTE技术。为了活下去，电信和联通显然不会坐以待毙。

根据网络上流传的消息，为了获得工信部的支持，移动将原本用于GSM覆盖的4MHz低频频谱转让给中国联通，用于联通的4G网络覆盖。一直以来，移动使用890-909MHz共计19MHz的对称频谱用于GSM网络的覆盖。而联通仅有909-915MHz共计6MHz的对称频谱用于GSM网络的覆盖。由于频谱极为有限，联通的GSM网络覆盖不仅远不如中国移动，也很难用于FDD-LTE网络的建设。假如消息属实，移动让出4MHz频谱给联通，那么对于低频严重不足的联通无异于是一剂良药。虽然10MHz对称频谱和移动的15MHz还有很大差距，但是用于FDD-LTE的建设已是足够。一旦低频4G网络建设完毕，联通就有了一张覆盖完善的4G基础网络，4G信号差也就成为了历史。

另一方面，对于移动超低价甚至免费宽带对电信、联通造成的冲击，移动也在2018年大幅降低了宽带网络的投资。一旦电信和联通的宽带现金流得到保存，两家运营商就有可能在激烈的竞争中存活下去。与此同时，由于电信VoLTE技术迟迟未能上马，电信的2G网络持续处于较高的负荷水平。由于频谱严重不足，电信还在持续向工信部申请821-825MHz的频谱资源（虽然申请了十年都未能通过）。假如这一申请可以获得移动的支持，那么对于电信可谓是雪中送炭。不难看出，假如移动可以和电信联通两家运营商实现某种默契，那么换取电信联通的支持也并非不可能

总结：

在“提速降费”的大背景下，移动迫切需要降低网络运维成本，留出资费的下降空间。为了解决这一问题，GSM和TD-SCDMA的清频退网就显得尤为重要。只有获得FDD-LTE牌照，才可以名正言顺的开始GSM的退网工作。另一方面，在竞争中处于绝对劣势的电信和联通显然并不同意移动获得FDD牌照，如何获得两家运营商的支持就显得尤为重要。无论结果如何，2G、3G网络的退网都将不可避免。
